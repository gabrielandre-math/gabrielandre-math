## Sobre Mim

Olá! Sou Gabriel André, Engenheiro de Software Full-Stack. Minha especialidade é a modernização de sistemas legados e a otimização de performance para aplicações de larga escala.

Atualmente, estou focado em um projeto complexo de migração de uma plataforma monolítica em **Java/Spring Boot** para uma arquitetura moderna de microsserviços com **.NET**. Este trabalho envolve não apenas o desenvolvimento de novas APIs e serviços, mas também a reengenharia de processos para garantir maior escalabilidade e eficiência.

Acredito que as melhores soluções técnicas nascem de um profundo entendimento do produto e do negócio.
